# Windows Server Active Directory Lab

Overview
Built a complete Windows Server environment on AWS to practice enterprise IT administration tasks.

Technologies Used
- AWS EC2 (t3.micro)
- Windows Server 2025
- Active Directory Domain Services
- Group Policy Management
- Zendesk (ticketing system)

Lab Architecture
- Domain: mobileglass.local
- Domain Controller: MobileGlass
- Users: 10+ across 4 departments
- Groups: 6 security groups
- OUs: Hierarchical structure (Corporate â†’ Departments)

Skills Demonstrated

Active Directory Administration
- Promoted server to Domain Controller
- Created organizational structure (OUs)
- Implemented user lifecycle management
- Configured group policies
- Password policy enforcement
- Account lockout policies

User Management Tasks Performed
- Created new user accounts  
- Disabled/deleted terminated users  
- Reset passwords  
- Modified group memberships  
- Locked and unlocked accounts  
- Created service accounts  

Troubleshooting & Problem Resolution
- Intentionally broke AD configurations
- Recovered deleted objects using AD Recycle Bin
- Documented all resolutions

Ticketing System
- Created and resolved 5 realistic IT tickets
- Used professional documentation standards
- Tracked resolution times
